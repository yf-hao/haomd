# Extract Contact Skill

这个 skill 用于从一段文本中提取联系人信息，并返回结构化结果。

## 适用场景

- 用户要求提取联系人信息
- 用户要求从文本中识别姓名、邮箱、电话
- 需要将联系人结果交给其它 workflow 步骤继续处理

## 使用原则

- 调用脚本前，先确认任务是“提取联系人信息”
- 调用时传结构化参数 `text`
- `text` 只包含需要处理的正文
- 不要把额外解释性指令一起传给脚本

## Scripts

### run

用途：

- 从 `text` 中提取联系人信息

何时使用：

- 当用户要求“提取联系人”“识别姓名电话邮箱”时

输入参数：

- `text`：待提取的正文文本

正确示例：

用户输入：

`请从这段话中提取联系人信息：张三，邮箱 zhangsan@example.com，电话 13800138000`

应提取参数：

```json
{
  "text": "张三，邮箱 zhangsan@example.com，电话 13800138000"
}
```

错误示例：

不要传：

```json
{
  "instruction": "请从这段话中提取联系人信息",
  "text": "张三，邮箱 zhangsan@example.com，电话 13800138000"
}
```

预期结果：

- 脚本成功执行时，返回 JSON 字符串
- JSON 中包含：
  - `name`
  - `email`
  - `phone`
