# Wrap Text Skill

这个 skill 用于把一段正文用指定前缀和后缀包起来。

## 适用场景

- 用户要求给文本加前后缀
- 用户要求把一句话包装成特定格式
- workflow 已经拿到结构化结果，需要生成最终展示文本

## 使用原则

- 调用脚本前，先提取结构化参数
- 必须传 `text`、`prefix`、`suffix`
- 不要把整句自然语言直接传给脚本

## Scripts

### run

用途：

- 把 `text` 按 `prefix + text + suffix` 形式输出

何时使用：

- 当用户明确要求“在前面加什么、后面加什么”
- 当 workflow 需要把上一步的结果包装成展示文本

输入参数：

- `text`：正文
- `prefix`：前缀
- `suffix`：后缀

正确示例：

用户输入：

`请把“张三”用【 和 】包起来`

应提取参数：

```json
{
  "text": "张三",
  "prefix": "【",
  "suffix": "】"
}
```

错误示例：

不要传：

```json
{
  "text": "请把张三用【和】包起来"
}
```

预期结果：

- 脚本成功执行时，返回包裹后的文本
- 输出应严格等于 `prefix + text + suffix`
